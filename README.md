# 🤖 Llama Groq Chat App

Aplicação web simples de chat utilizando **Streamlit** e o modelo de linguagem **Llama 3.3 (Groq)** via LangChain.

## 📌 Descrição

Este projeto implementa uma interface de chat interativa onde o usuário pode enviar mensagens e receber respostas geradas por um modelo de linguagem (LLM). O histórico da conversa é mantido durante a sessão.

## 🚀 Tecnologias utilizadas

- Python
- Streamlit
- LangChain
- Groq (modelo Llama 3.3)
- python-dotenv

## 📂 Estrutura

- `app.py` — Arquivo principal da aplicação
- `.env` — Arquivo de variáveis de ambiente (não versionado)
